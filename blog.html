CSS Flexbox হলো একমাত্রিক (কেবল সারি বা কেবল কলাম) লে-আউট মডিউল। অন্যদিকে, CSS Grid হলো দ্বিমাত্রিক (একই সাথে সারি ও কলাম) লে-আউট মডিউল।

Bootstrap একই সাথে Utility-ভিত্তিক ও Component-ভিত্তিক ফ্রন্টএন্ড ফ্রেমওয়ার্ক। অন্যদিকে, TailwindCSS হলো কেবল Utility-ভিত্তিক ফ্রন্টএন্ড ফ্রেমওয়ার্ক। কম্পোনেন্ট নিজে লিখে নিতে হবে বা থার্ড পার্টি কম্পোনেন্ট ব্যবহার করতে হবে। তবে বুটস্ট্র্যাপের তুলনায় টেইলউইন্ডে ইউটিলিটির সমাহার অনেক ব্যাপকতর।

প্রতিটা HTML Element একটি বক্সে মুড়োনো থাকে। সে বক্সে Content ঘিরে Padding আছে, Padding ঘিরে Border থাকে, আর Border ঘিরে Margin। এই ধারণাই CSS Box Model।

এলিমেন্টের লে-আউট ও ডিজাইনে এই বক্স মডেল একটি কেন্দ্রীয় অপরিহার্য ধারণা।

HTML-এ প্রতিটা Tag হয় Semantic Tag অথবা Non-Semantic Tag।

Semantic Tag কেবলমাত্র এর নামের মাধ্যমেই এর কনটেন্টের অর্থ বা পুরো পেজের (বা কমপক্ষে সিবলিং এলিমেন্ট বা এলিমেন্টদের) কনটেন্টের মাঝে নিজের কনটেন্টের অর্থগত ভূমিকা (Semantic Role) প্রকাশ করে। যেমন— form ও table এইসব ট্যাগ যথাক্রমে বুঝায় যে, এর কনটেন্ট একটি ফর্ম ও একটি টেবিল। আবার, h1 ট্যাগ বুঝায় যে, এর কনটেন্ট পুরো পেজের কনটেন্টের মাঝে বা কমপক্ষে এর সিবলিং এলিমেন্ট বা এলিমেন্টদের কনটেন্টদের মাঝে ১ম স্তরের শিরোনামের ভূমিকা পালন করে।

অন্যদিকে, Non-Semantic Tag যেমন— div এর কেবল নাম থেকে এর কনটেন্টের অর্থ বা (বৃহত্তর প্রেক্ষাপটে) অর্থগত ভূমিকা বুঝার উপায় নেই।
